# Worship Repository App

This is code repo for the Worship Repository App. Built with Svelte.

## Developing

Once you've cloned the project and installed dependencies with `npm install` (or `pnpm install` or `yarn`), start a development server:

```sh
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```
Always develop new features in a new branch.

## Building

To create a production version of your app:

```sh
npm run build
```

You can preview the production build with `npm run preview`.

## Deployment

To deploy changes, first create a pull request from feature branch to master branch. Once merged, the changes are automatically deployed to Firebase.
